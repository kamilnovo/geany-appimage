# Changelog

All notable changes to this project will be documented in this file.

## [v2.0-enhanced1] – 2026-02-21

### Added
- Fully portable Geany AppImage build
- Preconfigured Darcula theme
- JetBrains Mono font support
- TreeBrowser plugin enabled by default
- ColorPreview plugin enabled by default
- ExtraSel, LineOperations, MultiTerm, Markdown plugins enabled
- Custom toolbar layout
- Improved syntax highlighting (CSS, C, Python)
- Custom snippets and templates
- Minimap enabled
- Modern editor defaults (indentation, UTF‑8, LF)
- GitHub Actions automated build
- Automatic GitHub Releases on tag
- AppImage update metadata + `.zsync` delta updates
- English README

### Notes
This is the first public enhanced release of the portable Geany AppImage.
