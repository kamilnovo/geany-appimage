# Geany 2.0 AppImage (with Plugins + Color Preview)

This project builds a fully self-contained **AppImage** that includes:

- **Geany 2.0**
- **geany-plugins 2.0**
- the **Color Preview** plugin (with enlarged color squares patch)
- many additional useful plugins (see list below)
- a complete AppDir structure
- no system dependencies required

The AppImage runs on MX Linux 23/25, Debian, Ubuntu, Mint, Arch, Manjaro and many other distributions.
